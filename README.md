## Fitur Utama Aplikasi
1. **CRUD Data Transaksi:** Tambah, lihat, dan hapus riwayat pengeluaran.
2. **Dashboard Ringkasan:** Kalkulasi total pengeluaran dan jumlah transaksi secara real-time.
3. **Filter Kategori:** Menyaring riwayat berdasarkan kategori tertentu.
4. **Integrasi API Publik:** Menampilkan Live Kurs USD ke IDR menggunakan Exchange Rate API.

## Struktur Folder
- `frontend/` : Berisi berkas HTML, CSS, dan JavaScript Client-side.
- `backend/` : Berisi server logika Express.js.
- `database/` : Berisi konfigurasi dan berkas database SQLite (`keuangan.db`).
